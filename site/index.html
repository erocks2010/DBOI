Assumptions for this task

  • The Table Updates Continously and is always sorted by LastChangedBid value

Approach followed

  1. UI
    • Blank table element in html file
    • The data from STOMP subscription is read
    • If the data is new then it is added to currenciesInOrder and passed onto sortTable function
    • OR If the data is already present, then the data is passed onto sortTable function along with currenciesInOrder
    • The sortTable sorts the currenciesInOrder array by LastChangedBid column
    • The sorted currenciesInOrder array is passed to addSparklineDataToSortedTable where Sparkline data is added/updated on currenciesInOrder
    • The currenciesInOrder is then passed to drawTable function which clears the Table and then adds all the data to it and renders
  2. Testing
    • Mocha has been used for Testing.Run npm test to see the results
